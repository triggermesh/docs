    tmctl start

    tmctl start

    Starts TriggerMesh components

    tmctl start [broker] [flags]

    Examples

    tmctl start

    Options

      -h, --help      help for start
      --restart   Restart components

    Options inherited from parent commands

          --version string   TriggerMesh components version. (default "v1.25.1")
